# Pomodoro Timer
Um temporizador simples estilo Pomodoro. O usuário pode selecionar o tempo desejado, quando o tempo termina, um alarme é emitido.

### Link https://pomodorotimer-me.netlify.app/

### Tecnologias utilizadas:
- HTML
- CSS
- Javascript
